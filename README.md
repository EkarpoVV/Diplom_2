# Diplom_2
# Проект автоматизации тестирования приложения Stellarburgers, бэкенд часть.
1. Основа для написания автотестов — фреймворк pytest.
2. Установить зависимости — pip install -r requirements.txt.
3. Команда для запуска — pytest -v. 

# Проект состоит из шести груп тестов 

test_change_user_data.py - Проверяет изменение данных пользователя

test_create_order.py - Проверяет создание заказа

test_create_user.py - Проверяет создание пользователя

test_get_user_orders.py - Проверяет получение заказов пользователя

test_login_user.py - Проверяет логин пользователя
